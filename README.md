## Building
### macOS

You need `qemu` installed.

After cloning or downloading the source, navigate to the `run` folder in the terminal.
Then run: `qemu-system-x86_64 kernel.bin`
Or if just `qemu` is in your PATH: `qemu kernel.bin`

The OS will load.
